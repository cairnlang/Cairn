# Runewarden Book Code

This tree will hold chapter-by-chapter Cairn source for the tutorial book.

## Intended Layout

- `runewarden/chapters/chXX_*`: isolated chapter snapshots.
- `runewarden/final/`: the full capstone version (added later).
- `runewarden/shared/`: shared fixtures/data when chapter snapshots start to need them.

For now, chapter code has not been authored yet; this directory is prepared so we can start writing chapter artifacts incrementally.
